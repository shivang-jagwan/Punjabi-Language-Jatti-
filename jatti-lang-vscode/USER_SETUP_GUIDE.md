# Jatti Language - User Setup Guide

## Installation Steps

### Step 1: Install VS Code Extension

1. Open VS Code
2. Go to Extensions (`Ctrl+Shift+X`)
3. Search: **"Jatti Language"**
4. Click **Install**

### Step 2: Download Jatti CLI Files

Option A: **From GitHub** (Recommended)
- Download: https://github.com/your-username/jatti-lang
- Extract to your workspace folder

Option B: **Manual Setup**
